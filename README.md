# Modellix Agent Skill

An Agent Skill that enables AI agents to access 100+ generative AI models on the Modellix platform, facilitating a wide range of AI-powered content creation and manipulation tasks.

## What is Modellix?

[Modellix](https://modellix.ai) is a unified Model as a Service (MaaS) platform that provides API access to over 100 AI models for generative AI tasks. Agents can leverage Modellix to:

- **Generate images from text**: Utilizing models like Qwen Image, Wan 2.6 T2I, and Seedream series.
- **Create videos from text or images**: Employing models such as Seedance 1.5 Pro, Wan 2.6 T2V, and Hailuo 02.
- **Edit existing images**: Including capabilities like style transfer, object addition/removal, color adjustment, and precise bilingual text editing.
- **Perform virtual try-ons**: Generating AI virtual try-on effects for clothing and fashion.
- **Translate text within images**: Across 11 languages with Qwen MT Image.
- **Specialized Image Generation**: Such as Wordart creation, sketch-to-image conversion, background generation, and image outpainting.

Modellix supports asynchronous task processing with transparent pricing, detailed logging, and enterprise-grade reliability.
