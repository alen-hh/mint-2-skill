# Modellix Agent Skill

An Agent Skill that enables AI agents to access 100+ generative AI models on the Modellix platform, facilitating a wide range of AI-powered content creation and manipulation tasks.

## What is Modellix?

[Modellix](https://modellix.ai) is a unified Model as a Service (MaaS) platform that provides API access to over 100 AI models covering generative AI tasks. Agents can leverage Modellix to:

- **Generate Images**: Create high-quality images from text prompts using models like Qwen Image, Wan 2.6 T2I, and Seedream series.
- **Transform Images**: Edit and transform existing images using text prompts, including style transfer, object addition/removal, and color adjustment.
- **Generate Videos**: Create videos from text descriptions or images with cinematic quality, supporting multi-shot narratives and custom audio.
- **Edit Images**: Perform specialized image editing tasks such as text editing in images (bilingual support), color enhancement, style transfer, object manipulation, and background generation.
- **Virtual Try-On**: Generate AI virtual try-on effects for clothing and fashion applications.
- **Translate Image Text**: Translate text within images across 11 languages.
- **Specialized Image Generation**: Create artistic text (Wordart), convert sketches to images, and extend images with outpainting.

Modellix offers asynchronous task processing with transparent pricing, detailed logging, and enterprise-grade reliability.
