# Modellix Agent Skill

An Agent Skill that enables AI agents to access 100+ generative AI models on the Modellix platform, covering a wide range of AI-powered content creation and manipulation tasks.

## What is Modellix?

[Modellix](https://modellix.ai) is a unified Model as a Service (MaaS) platform that provides API access to over 100 AI models. It aggregates cutting-edge generative AI models from leading providers such as Alibaba, ByteDance, and MiniMax, enabling agents to perform a diverse set of tasks including:

- **Text-to-Image Generation**: Create high-quality images from text prompts using models like Qwen Image, Wan 2.6 T2I, and Seedream series.
- **Image-to-Image Transformation**: Edit and transform existing images using text prompts, with capabilities for style transfer, object addition/removal, and color adjustment.
- **Text-to-Video Generation**: Generate videos from text descriptions with cinematic quality using models like Seedance 1.5 Pro and Wan 2.6 T2V.
- **Image-to-Video Generation**: Convert static images into video sequences with motion and animation using models like Wan 2.6 I2V and Seedance.
- **Image Editing Operations**: Perform specialized tasks such as text editing in images (bilingual support), color enhancement, style transfer, object manipulation, and background generation.
- **Virtual Try-On and Fashion**: Generate AI virtual try-on effects for clothing applications.
- **Image Translation**: Translate text within images across multiple languages.
- **Specialized Image Generation**: Create artistic text (Wordart), convert sketches to images, and generate backgrounds.

Modellix supports asynchronous task processing with transparent pricing, detailed logging, and enterprise-grade reliability, backed by extensive IT service expertise.
