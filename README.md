# Modellix Agent Skill

An Agent Skill that enables AI agents to leverage the comprehensive capabilities of the Modellix platform, providing API access to over 100 generative AI models for a wide range of tasks.

## What is Modellix?

[Modellix](https://modellix.ai) is a unified Model as a Service (MaaS) platform offering API access to a vast array of AI models from leading providers such as Alibaba, ByteDance, and MiniMax. It empowers agents to perform sophisticated generative AI tasks with enterprise-grade reliability and transparent pricing.

Key capabilities include:

*   **Generative AI**: Create images from text, generate videos from text or images, and perform advanced image editing.
*   **Specialized AI Tasks**: Virtual try-ons, image outpainting, text translation within images, and artistic WordArt creation.
*   **Model Variety**: Access to 100+ AI models including Qwen Image, Wan series, Seedream series, Seedance, Hailuo, and more.
*   **Asynchronous Processing**: Efficient handling of long-running generation tasks with task ID tracking.
*   **Enterprise Features**: Detailed logging, transparent pricing, and robust infrastructure for high availability.
