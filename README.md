# Modellix Agent Skill

An Agent Skill that enables AI agents to access 100+ generative AI models on the Modellix platform, providing a unified Model as a Service (MaaS) experience.

## What is Modellix?

[Modellix](https://modellix.ai) is a comprehensive Model as a Service (MaaS) platform offering API access to over 100 AI models for a wide range of generative AI tasks. Agents can leverage Modellix for:

- **Text-to-Image Generation**: Create high-quality images from text prompts using models like Qwen Image, Wan 2.6 T2I, and Seedream series.
- **Image-to-Image Transformation**: Edit and transform existing images with text prompts, including style transfer, object manipulation, and color adjustments.
- **Text-to-Video Generation**: Generate videos from text descriptions with cinematic quality using models like Seedance 1.5 Pro and Wan 2.6 T2V.
- **Image-to-Video Generation**: Convert static images into animated video sequences with models like Wan 2.6 I2V and Seedance.
- **Image Editing Operations**: Perform specialized tasks such as bilingual text editing, color enhancement, style transfer, object addition/removal, image outpainting, and background generation.
- **Virtual Try-On**: Generate AI virtual try-on effects for fashion applications.
- **Image Translation**: Translate text within images across multiple languages.
- **Specialized Image Generation**: Create WordArt, convert sketches to images, and generate backgrounds.

The platform supports asynchronous task processing, transparent pricing, detailed logging, and enterprise-grade reliability.
